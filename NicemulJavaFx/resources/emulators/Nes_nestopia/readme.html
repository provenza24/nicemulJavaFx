Introduction

Nestopia est un émulateur open source NES/Famicom écrit en C++. Il a été crée pour être aussi précis que possible. Le développement a débuté dans la moitié de de l'année 2002, pour une sortie sous Windows, un an plus tard. Il a depuis été porté sur d'autres plateformes, incluant Linux et Mac OS X. Consultez la section "Crédits" pour connaitre la liste des auteurs.

Troubleshooting

Si vous avez des problèmes de ralentissement avec Nestopia, voici quelques conseils :

  • Diminuer la résolution de lécran et/ou la qualité vidéo (dans options ==> vidéo).
  • Désactiver les filtres vidéo (dans options ==> vidéo).
  • Changer la localisation de la mémoire de l'écran de la NES (dans options ==> vidéo)
  • Activer le Triple Buffering (dans options ==> timing).
  • Activer le saut dimage auto (dans options ==> timing).
  • Désactiver la vsync (dans options ==> timing).
  • Désactiver le rembobineur (dans options ==> timing).
  • Changer la priorité de lapplication en passant à "au dessus de la normal" ou "haute" (dans options ==>préférences)
  • Diminuer la fréquence du son et/ou léchantillonnage ou désactiver la sortie son complètement (dans options ==> son).
  • Si votre pilote/driver de carte vidéo a une option pour la VSync, désactivez là.

Si vous avez des problèmes pour lancer un jeu en particulier, voici quelques pistes :

  • Cest un mauvais dump.
  • L'image ne comportae pas suffisament d'information sur l'émulation ou est incorrecte.
  • Fonctionne-t-il sur la console ? Si ce n'est pas le cas, il n'est pas supposé fonctionner avec Nestopia aussi.
  • Il a été hacké pour fonctionner seulement sur un émulateur en particulier.
  • La région n'est pas correctement paramètré, essayez de passer en PAL et vice versa.
  • Le contrôleur approprié est connecté ? Par exemple, cest un programme qui pourrait fonctionner quavec le clavier ou un jeu qui ne fonctionne quavec le second contrôleur.
  • Est-ce que l'option 'Aucune limitation de sprite' est activé ? Essayez de la désactiver, certains jeux peuvent dépendre de cette option à cause du matériel.
  • Il peut y avoir des erreurs dans la Base de Donnée. Essayez de l'arrêter temporairement.
  • Il nest pas supporté par Nestopia.

Divers :

  • Si vous avez un problème avec l'affichage de ligne dans le lanceur, veuillez consulter ce lien.
